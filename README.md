# Smoking-Dataset-Analysis-Dashboard

# Overview
This project is an analytical dashboard for smoking and medical cost data, designed to explore the relationship between smoking behavior and key factors such as gender, age, number of children, and region, along with analyzing its impact on medical expenses.

# Overview
This project is an analytical dashboard for smoking and medical cost data, designed to explore the relationship between smoking behavior and key factors such as gender, age, number of children, and region, along with analyzing its impact on medical expenses.

# Key Metrics (KPIs)
Total Sample Size: 216 individuals
Non-Smokers: 176 (81.48%)
Smokers: 40 (18.52%)

# Financial Analysis
 Average Medical Charges (Overall):
Non-Smokers: 39.51K (73.61%)
Smokers: 14.09K (26.24%)
 Average Charges by Gender:
Male: 19.6K (52.09%)
Female: 18.02K (47.91%)

# Smoking Analysis
 Male smokers: 110 (50.93%)
Female smokers: 106 (49.07%)
 Overall Distribution:
The majority of the dataset are non-smokers
Smokers represent less than 20% of total data

# Key Insights
A very high percentage of the dataset are non-smokers (over 80%)
No significant difference between males and females in smoking behavior
Medical expenses are relatively similar across genders
Number of children has a noticeable impact on smoking patterns
Slight variation in smoking distribution across regions

# Dashboard Features
 Interactive filters (Gender – Age Range – BMI)
 Multiple visualizations (Pie Charts, Bar Charts, KPIs)
 Multi-dimensional analysis (Region / Children / Gender)
📈 Clear representation of relationships between smoking and medical costs

Power BI

This dashboard provides a comprehensive view of smoking behavior and its impact, helping to:
